ApexSensitivity_FULL_Fabric_1.21.1

Этот проект представляет собой мод для Minecraft Fabric версии 1.21.1, который полностью управляет настройками чувствительности и профилями игрока. Мод предназначен для улучшения игрового опыта, предоставляя удобный интерфейс и гибкие возможности настройки под любого игрока.

Основные возможности:
- Управление чувствительностью мыши
- Создание и сохранение профилей игрока
- Интеграция с модом Mod Menu для быстрого доступа к настройкам
- Визуальные элементы интерфейса (HUD) для отображения активного профиля
- Поддержка всех стандартных элементов Fabric

Установка:
1. Скачайте и установите Fabric Loader версии 1.21.1
2. Скопируйте jar-файл мода в папку mods вашего Minecraft
3. Запустите игру через Fabric

Использование:
- Настройки мода доступны через интерфейс Mod Menu
- Для изменения чувствительности и профилей используйте HUD и GUI мода
